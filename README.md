1. Introdução
Este relatório descreve a implementação da primeira fase do tutorial de Computação Móvel, focada na aprendizagem da linguagem Kotlin através do IntelliJ IDEA. O foco principal foi o desenvolvimento de lógica de programação, controlo de fluxo e a aplicação de conceitos avançados de Programação Orientada a Objetos (POO).

2. Visão Geral do Sistema
O projeto consiste em:

Exercícios de Lógica: Resolução de problemas com arrays, estruturas de decisão e repetição em Kotlin.

Sistema de Biblioteca Virtual: Uma aplicação de consola que demonstra o uso de herança e classes abstratas para gerir livros físicos e digitais.

3. Arquitetura e Design
A biblioteca foi desenhada para ser extensível. A classe Livro serve como base para diferentes formatos, utilizando métodos abstratos para garantir que cada tipo de livro forneça as suas informações de armazenamento específicas.

4. Implementação
Custom Getters/Setters: Validação de stock em tempo real para evitar valores negativos.

Companion Objects: Implementação de um contador global para rastrear o número total de obras no sistema.

Data Classes: Uso de classes de dados para uma representação limpa dos membros da biblioteca.
